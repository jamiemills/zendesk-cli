# TicketQ Zendesk Adapter

Zendesk adapter plugin for TicketQ - Universal ticketing system CLI and library.

This adapter provides integration with Zendesk's REST API for unified ticket management.

## Installation

```bash
pip install ticketq-zendesk
```

## Configuration

```bash
tq configure --adapter zendesk
```

## Usage

```bash
# List tickets
tq tickets

# Configure Zendesk
tq configure --adapter zendesk
```

## Features

- Complete Zendesk API integration
- Secure credential storage
- Team name resolution
- Status normalisation
- Multi-status filtering

For more information, see the main TicketQ documentation.
